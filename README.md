## how to load into chrome
1. open chrome://extensions/ in your chrome
1. enable developer mode (right top corner)
1. `Load Unpacked`
1. choose this folder

## download jquery
```bash
wget https://code.jquery.com/jquery-3.4.1.min.js
```
